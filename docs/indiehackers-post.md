# I built an eCPM comparison dashboard for mini-game developers. Looking for 3 beta testers.

I am building a small tool called eCPM Bazaar.

The problem came from a very practical workflow: if you monetize a mini-game or mobile app with ads, you often need to check multiple dashboards just to answer a basic question:

**Why did my ad revenue change yesterday?**

Was it because:

- eCPM dropped?
- impressions dropped?
- fill rate changed?
- one country performed worse?
- one ad source started underperforming?

For a small team, manually checking this every day is painful.

The first version of eCPM Bazaar connects to the TopOn reporting API and shows:

- Revenue
- weighted eCPM
- impressions
- fill rate
- app / placement / country / ad source breakdowns
- eCPM, revenue, and fill-rate drop alerts
- demo mode and real-data mode

The current product status is honest:

- TopOn API integration is working.
- My own test TopOn account is new and has no monetization records yet.
- So I am looking for a few developers with real ad data to test whether the dashboard is actually useful.

I am looking for 3 beta testers who:

- have a mobile game, mini-game, or app with ad monetization
- use TopOn or another mediation/reporting platform
- have recent ad impressions or revenue data
- want a clearer view of eCPM and fill-rate changes

Security boundary:

- I do not need your account password.
- Read-only reporting access is preferred.
- Your app name, screenshots, revenue, and raw data will not be shared publicly.
- You can revoke access after the test.

What I can provide for free:

- a short eCPM / revenue / fill-rate diagnosis
- country and placement level breakdowns
- alert rule suggestions
- feedback on which monetization metrics are worth monitoring daily

I am not claiming this will magically increase revenue. The goal is simpler: make ad monetization data easier to compare, diagnose, and act on.

If you are building or running an ad-monetized app/game and want to test it, leave a comment or DM me.

Also curious: if you monetize with ads, what is more painful for you right now: eCPM, fill rate, mediation setup, or understanding revenue drops?
